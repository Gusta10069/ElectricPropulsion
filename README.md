# ElectricPropulsion
A project for code related to propulsion for aerospace stuff.

## 3D world demo
You can run the Three.js demo locally or inside Codex without using Git commands directly.

### Run in Codex
1. Open a terminal in this repo (already checked out here).
2. Start a simple web server:
   ```bash
   python -m http.server 8000
   ```
3. Open the preview/browser to:
   ```
   http://localhost:8000/index.html
   ```

### Run on your machine
1. Clone or pull the repo, then `cd` into it.
2. Start the same server:
   ```bash
   python -m http.server 8000
   ```
3. Visit `http://localhost:8000/index.html`.
